Perancangan dan Implementasi Sistem Analisis Sentimen Ulasan Aplikasi Google Play Store Berbasis IndoBERT dan BERTopic

Proyek ini merancang dan mengimplementasikan sistem analisis sentimen ulasan aplikasi Google Play Store berbahasa Indonesia menggunakan model IndoBERT untuk klasifikasi sentimen dan BERTopic untuk analisis topik, yang mencakup proses pengumpulan data otomatis, preprocessing teks, pelatihan model deep learning, serta visualisasi hasil dalam antarmuka web berbasis Flask. Selain melakukan klasifikasi sentimen konvensional, sistem ini juga dilengkapi dengan tiga fitur unik, yaitu (1) Confidence Scoring dan Entropy Analysis untuk mengukur tingkat keyakinan serta ketidakpastian prediksi model, (2) 4-Component Ambiguity Detection yang digunakan untuk mengidentifikasi ulasan dengan makna ganda atau kontradiktif, serta (3) Aspect-Based Sentiment Analysis (ABSA) dengan delapan aspek domain-spesifik guna menganalisis sentimen secara lebih rinci berdasarkan aspek yang dibahas oleh pengguna.
